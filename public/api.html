Documentação da API

Endpoints Disponíveis

GET /nicks/generate?nick=string

Gera todas as variações de fonte para o nickname fornecido

Exemplo de Requisição:
fetch('/nicks/generate?nick=Player')
.then(response => response.json())
.then(data => console.log(data));
Exemplo de Resposta:
{
"original": "Player",
"fonts": [
{
"font": "boldSerif",
"nick": "𝐏𝐥𝐚𝐲𝐞𝐫"
},
...
]
}
GET /nicks/decorated?nick=string

Retorna nicks com decorações especiais

Exemplo de Requisição:
fetch('/nicks/decorated?nick=Player')
.then(response => response.json())
.then(data => console.log(data));
Exemplo de Resposta:
{
"original": "Player",
"variants": [
"꧁༺Player༻꧂",
"𖧄 𝐏𝐥𝐚𝐲𝐞𝐫 𖧄",
...
]
}
